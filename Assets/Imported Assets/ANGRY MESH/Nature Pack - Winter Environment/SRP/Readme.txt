For both HDRP and URP templates you need to use Unity version 2021.3.35f1 or newer. 

--------------------------------------------------------------------------------

HDRP - HOW TO USE:
1. Create a new empty project using HDRP Template.
2. Import the Winter Environment package into the project.
3. Import the HDRP Template from ANGRY MESH > Nature Pack - Winter Environment > SRP. 
- For Unity 6000.0.23 or newer, HDRP version 17.0.3 or above - You need to use HDRP template 17.0.3.unitypackage
- For Unity 2022.3.18, HDRP version 14.0.10 or above - You need to use HDRP template 14.0.10.unitypackage
- For Unity 2021.3.35, HDRP version 12.1.14 or above - You need to use HDRP template 12.1.14.unitypackage

--------------------------------------------------------------------------------

URP - HOW TO USE:
1. Create a new empty project using URP Template.
2. Import the Winter Environment package into the project. 
3. Import the URP Template from ANGRY MESH > Nature Pack - Winter Environment > SRP. 
- For Unity 6000.0.23 or newer, URP version 17.0.3 or above - You need to use URP template 17.0.3.unitypackage
- For Unity 2022.3.18, URP version 14.0.10 or above - You need to use URP template 14.1.10.unitypackage
- For Unity 2021.3.35, URP version 12.1.14 or above - You need to use URP template 12.1.14.unitypackage

--------------------------------------------------------------------------------
